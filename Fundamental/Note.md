2. ファンダメンタル分析のポイント
    - 成長
        - EPS (Earning Per Share)が25%以上成長している株
        - 過去3年において毎年収益が25%以上の株
        - ROE (Return on equity)が15%以上の成長
            - EPS = 当期純利益 / 発行済株数
            - BPS = 純資産 / 発行済株数
            - ROE = EPS / BPS = 当時純利益 / 純資産
    - 割安
        - P/E　-> 割安かどうかを判断する。
            - PE < 20以下は望ましい
            - PE = 株価 / 一株あたりの利益
    - 安定 
        - フローキャッシュフローがある。
        - 配当
    - 役員は大量に売る時には注意。
    - 業界、製品
        - 産業をリードしている製品やサービスがあるのか?
    - 出来高の増加は隠しきれない人気の証拠
    - 上昇している株を投資する。 
    - 保有する株は5個程度 
    - ブラック・スワンを避ける
        - ブブルの影響を最も受けるのは、金持ちではなく、金持ちになりたがっている人
